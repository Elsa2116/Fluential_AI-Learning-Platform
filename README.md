# AI Learning Platform (Round 2)

## Project Structure

- `backend/` — FastAPI backend **folder structure only** (no code yet)
- `frontend/` — Next.js-style frontend scaffold

## Team Workflow

- Use a single shared GitHub repository.
- Create feature branches per task.
- Open pull requests and review before merging to `main`.

## Suggested Branch Naming

- `feature/backend-auth`
- `feature/backend-ai-tutor`
- `feature/frontend-dashboard`

## Quick Start (Scaffold)

1. Backend developer works in `backend/`
2. Frontend developer works in `frontend/`
3. Sync API contracts via pull requests
